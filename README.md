# Todo app created using FastAPI

## Requirements
 - python3.10+ 
 - poetry

# Setup

### Clone git repository

```bash
git clone https://github.com/iamnovichek/FastAPI_todo.git
```

### Go to FastAPI_todo

```bash
cd FastAPI_todo
```

### Intall all requirements

```bash
poetry init
```

# Run app

```bash
uvicorn main:app --reload
```
